# ユーザーマニュアル

## 使い方

1. トップ画面を開きます。
2. 変換したいファイルを 1 つ選択するか、ドロップ領域へドラッグします。
3. `変換する` を押します。
4. 変換結果が `変換結果` パネルに表示されます。
5. 必要に応じて `コピー` または `ダウンロード` を使います。

## 画面の各項目

- 対応拡張子: 現在の設定でアップロード可能な拡張子一覧です。
- サイズ上限: アップロードできる最大サイズです。
- コピー: 表示された Markdown をクリップボードへコピーします。
- ダウンロード: 表示された Markdown を `.md` ファイルとして保存します。

## エラー時の見方

- エラー時は、要約、考えられる原因、対処方法を画面に表示します。
- 対応外の拡張子を送ると、対応形式を確認するメッセージが表示されます。
- サイズ上限を超えた場合は、上限超過のメッセージが表示されます。
- Python や MarkItDown の実行に失敗した場合は、環境確認用のヒントが表示されます。
- コピーに失敗した場合は、ブラウザー権限の確認やダウンロード利用を促すメッセージが表示されます。
- サーバー内部エラーでは、サーバーが返した要約を優先して表示し、原因候補や対処方法がない欄は表示しません。

## 代表的なエラー

- ファイル未選択: ファイルを選び直してください。
- 非対応拡張子: 対応拡張子一覧を確認してください。
- サイズ超過: ファイルを小さくするか、設定上限を見直してください。
- Python 実行環境不備: Python のパスとインストール状態を確認してください。
- MarkItDown 未導入: 対象 Python 環境へ `markitdown` をインストールしてください。
- 変換失敗: 元ファイルの破損や追加依存関係不足を確認してください。

## 対応形式の考え方

既定では、日常的に扱うことが多い形式に絞っています。

- 文書: `.pdf`, `.docx`, `.pptx`, `.xlsx`
- テキスト系: `.txt`, `.md`, `.csv`, `.json`, `.xml`, `.html`, `.htm`
- 画像: `.jpg`, `.jpeg`, `.png`

必要なら設定ファイルで拡張できます。

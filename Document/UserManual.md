# ユーザーマニュアル

## 基本操作

1. 変換したいファイルを 1 つ選択します。
2. `変換する` を押します。
3. 変換結果を確認します。
4. 必要に応じて `コピー` または `ダウンロード` を使います。

## エラー表示の見方

エラー時は画面に次の情報が表示されます。

1. エラーの要約
2. 考えられる原因
3. 対処方法

### 代表的なエラー

- ファイル未選択: ファイルを選び直してください。
- 非対応拡張子: 対応拡張子一覧を確認してください。
- サイズ超過: ファイルを小さくするか、設定上限を見直してください。
- Python 実行環境不備: Python のパスとインストール状態を確認してください。
- MarkItDown 未導入: 対象 Python 環境へ `markitdown` をインストールしてください。
- 変換失敗: 元ファイルの破損や追加依存関係不足を確認してください。

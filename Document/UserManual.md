# ユーザーマニュアル

## 使い方

1. トップ画面を開きます。
2. 変換したいファイルを 1 つ選択するか、ドロップ領域へドラッグします。
3. `変換する` を押します。
4. 変換結果が `変換結果` パネルに表示されます。

## 画面の各項目

- 対応拡張子: 現在の設定でアップロード可能な拡張子一覧です。
- サイズ上限: アップロードできる最大サイズです。
- コピー: 表示された Markdown をクリップボードへコピーします。
- ダウンロード: 表示された Markdown を `.md` ファイルとして保存します。

## エラー時の見方

- 対応外の拡張子を送ると、対応形式を確認するメッセージが表示されます。
- Python や MarkItDown の実行に失敗した場合は、環境確認用のヒントが表示されます。

## 対応形式の考え方

既定では、日常的に扱うことが多い形式に絞っています。

- 文書: `.pdf`, `.docx`, `.pptx`, `.xlsx`
- テキスト系: `.txt`, `.md`, `.csv`, `.json`, `.xml`, `.html`, `.htm`
- 画像: `.jpg`, `.jpeg`, `.png`

必要なら設定ファイルで拡張できます。
